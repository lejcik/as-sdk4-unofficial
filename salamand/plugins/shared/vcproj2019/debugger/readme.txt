VC2019
======

VC2019 uses .natvis files.

VC2019: Copy altap.natvis file from subdirectory vc2019 to
C:\Program Files (x86)\Microsoft Visual Studio\2019\Community\Common7\Packages\Debugger\Visualizers

When you start new debugging session in VC2019 this file will be
loaded and you can enjoy properly shown data in ALTAP's data arrays (TDirectArray and
TIndirectArray, see arraylt.h in Shared directory).
